Intitulé de la formation

Formation Angular

Présentation

Dans ce cours, vous apprendrez rapidement à créer les composantes de base d'une application Angular. Vous apprendrez également à faire communiquer votre application Angular avec un serveur afin de créer une application web complète. Format/Durée : 32h sur 4 jours. La formation se fait en présentiel, avec un suivi individuel et un examen d’évaluation en fin de formation.

Environnement de travail

Suivi par un formateur-expert à toutes les étapes de la formation. Bureaux en centre-ville de Toulouse pour les parties présentielles. Matériel informatique fourni.

Audience

Développeurs/architectes, avec des bases de Javascript, HTML et CSS

Tarif

2500€ HT

Objectifs pédagogiques

  • Création d’applications web
  • Apprendre le langage TypeScript
  • Création de projet sur Angular
  • Utiliser des templates, le databinding, les directives
  • Créer des services Angular
  • Mettre en œuvre un système de routing
