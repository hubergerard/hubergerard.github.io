Vue qu'est-ce que c'est ?

Vue.js est devenu en quelques années le 3ème projet sur Github, toutes technologies confondues.

Devançant React et Angular, Vue.js est le framework js Open source le plus populaire. Avec cette formation, vous découvrirez cette librairie et son écosystème, et apprendrez les bonnes pratiques du développement JS moderne.
